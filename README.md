## üéì GLOSSARY OF TERMS

**[ControlNet](https://github.com/lllyasviel/ControlNet)** ‚Äì Extension for [Stable Diffusion](https://stability.ai/stable-diffusion) enabling pose, depth, and edge guidance  
**DAW (Digital Audio Workstation)** ‚Äì Professional audio editing software (e.g., Logic, Ableton)  
**Diffusion Model** ‚Äì AI architecture using iterative denoising to generate images/video  
**Inpainting** ‚Äì Filling or editing specific regions of an image/video  
**Latent Space** ‚Äì Compressed representation where AI models operate  
**LoRA (Low-Rank Adaptation)** ‚Äì Lightweight fine-tuning method for custom styles  
**NeRF (Neural Radiance Fields)** ‚Äì 3D scene reconstruction from 2D images  
**Outpainting** ‚Äì Extending images beyond original boundaries  
**Stem Separation** ‚Äì Isolating individual instruments/vocals from mixed audio  
**T2I (Text-to-Image)** ‚Äì Generating images from text descriptions  
**T2V (Text-to-Video)** ‚Äì Generating video from text descriptions  
**TTS (Text-to-Speech)** ‚Äì Converting written text to spoken audio  
**VST (Virtual Studio Technology)** ‚Äì Plugin format for audio software integration
